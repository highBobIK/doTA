# Учебный проект Dota 2 🎮

Привет! Это базовая структура для твоего будущего сайта по Dota 2. Я подготовил её так, чтобы тебе было удобно изучать веб-разработку и постепенно добавлять новые фишки.

## Что внутри?

*   `index.html` — это "скелет" твоего сайта. Здесь находится весь текст и структура (меню, заголовки, карточки героев).
*   `css/style.css` — это "внешний вид". Здесь задаются цвета (сейчас стоит темная тема с красными акцентами), шрифты и расположение элементов.
*   `js/script.js` — это "поведение". Здесь написан простой код для плавной прокрутки при клике на меню.

## Что ты можешь сделать дальше (задания для тренировки):

1.  **Добавить картинки:**
    В файле `index.html` найди блоки `<div class="hero-image placeholder" data-name="Pudge"></div>`.
    Замени их на настоящий тег картинки: `<img src="ссылка_на_картинку" class="hero-image" alt="Pudge">`. Картинки нужно будет сохранить в папку `img` (создай её!).
    Потом в CSS настрой размеры картинки (`width: 100%; height: 200px; object-fit: cover;`).

2.  **Добавить новые секции:**
    Посмотри, как сделана секция `<section id="heroes">` в `index.html`. Попробуй по аналогии создать секцию `<section id="news">` и добавить туда пару блоков с новостями.

3.  **Изменить цвета:**
    В файле `style.css` в самом верху есть блок `:root`. Там прописаны переменные цветов (например, `--accent-color`). Попробуй поменять код цвета `#c23c2a` на какой-нибудь другой и посмотри, как изменится сайт!

4.  **Фоновая картинка:**
    В `style.css` найди класс `.hero`. Сейчас там просто тёмный градиент. Попробуй добавить туда фоновую картинку с помощью свойства `background-image: url('путь_к_картинке');`.

Удачи в изучении создания сайтов! Если что-то сломается — не страшно, это нормальный процесс обучения.
